# Human–AI Cognitive Interface — Índice da Documentação (pt-BR)

Esta pasta contém uma documentação **ampla, coerente e altamente estruturada** derivada do diretório `info/`, que é a **Fonte de Verdade primária** deste repositório.

Princípios operacionais desta documentação (conforme a Fonte de Verdade):
- Não introduz novos requisitos, conceitos ou decisões.
- Reorganiza, consolida e reescreve apenas para clareza/estrutura, preservando rastreabilidade conceitual.
- Onde houver conflito entre fontes, o conflito é **registrado sem resolução**.
